# PROMPT.md - Analysis Templates

_Reusable templates for inflation-growth analysis tasks._

## /analyze

Perform full inflation-growth correlation analysis:

```
1. Read /home/node/.openclaw/shared/data/processed/daily-brief.md
2. Extract key inflation metrics (CPI, PCE, Core, PPI)
3. Identify correlations with:
   - Monetary policy indicators (Fed rates, balance sheet)
   - Labor market data (unemployment, wages)
   - Financial markets (yields, equities, sectors)
4. Assess current inflation regime and trajectory
5. Flag risk factors and watch items
6. Write structured analysis to (always overwrite, run every time when asked):
   /home/node/.openclaw/shared/analysis/macro-fundamentals/inflation-growth-analysis.md
```

## /quick-scan

Brief summary of latest inflation signals:

```
1. Read daily brief
2. Extract top 5 inflation-related data points
3. Note any significant changes from prior readings
4. Output 3-5 bullet summary
```

## /correlation-check

Deep dive on specific correlation:

```
1. Read daily brief
2. Focus on [SPECIFIED_CORRELATION]:
   - inflation-yields
   - inflation-wages
   - inflation-growth
   - inflation-equities
3. Analyze current relationship vs historical norms
4. Identify any regime shifts or anomalies
```

## /risk-assessment

Inflation risk factor analysis:

```
1. Read daily brief
2. Identify upside and downside inflation risks
3. Assess probability and impact of each
4. Rank by urgency
5. Provide monitoring recommendations
```
