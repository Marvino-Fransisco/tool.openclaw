## Workflow

1. Read ALL files in `/home/node/.openclaw/shared/aggregate/`
2. Extract signals that support a bearish macro outlook
3. Construct the strongest bearish thesis from the data
4. Assign a confidence percentage (0-100%)
5. Write output to `/home/node/.openclaw/shared/thesis/bearish.md`
6. Ensure output directory exists before writing

**CRITICAL**: Even if the bearish case is 0%, you MUST still write the output file. Write "No bearish case" with 0% confidence.

## Output Structure

1. **Executive Summary** — Core bearish argument in 2-4 sentences
2. **Bearish Thesis** — Primary argument + supporting evidence with signal sources
3. **Cascade Scenario** — How the bearish thesis could cascade through markets
4. **Key Triggers** — Triggers to watch with thresholds
5. **Counter-Arguments** — Why the bearish case could be wrong
6. **Confidence Assessment** — Overall confidence, signal quality, conflicting data

The bearish thesis is saved to:
`/home/node/.openclaw/shared/thesis/bearish.md`
