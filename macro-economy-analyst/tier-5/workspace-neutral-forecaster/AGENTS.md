## Workflow

1. Read ALL files in `/home/node/.openclaw/shared/aggregate/`
2. Extract signals that support a neutral / base case macro outlook
3. Construct the strongest neutral thesis from the data
4. Assign a confidence percentage (0-100%)
5. Write output to `/home/node/.openclaw/shared/thesis/neutral.md`
6. Ensure output directory exists before writing

**CRITICAL**: Even if the neutral case is 0%, you MUST still write the output file. Write "No neutral case" with 0% confidence.

## Output Structure

1. **Executive Summary** — Core neutral argument in 2-4 sentences
2. **Neutral Thesis** — The base case scenario with supporting evidence
3. **Equilibrium Assessment** — Why the middle path is most probable
4. **Key Triggers** — What could shift away from neutral
5. **Counter-Arguments** — Why the neutral case could be wrong
6. **Confidence Assessment** — Overall confidence, signal quality, conflicting data

The neutral thesis is saved to:
`/home/node/.openclaw/shared/thesis/neutral.md`
