# TOOLS.md - Local Notes

## Workflow Script

**Location:** `scripts/start-workflow.ts`

**Usage:**
```bash
npx tsx scripts/start-workflow.ts "<prompt>" [--timeout <ms>]
```

**Arguments:**
| Arg | Required | Default | Description |
|-----|----------|---------|-------------|
| `<prompt>` | Yes | - | Topic/question to research |
| `--timeout` | No | 300000 | Timeout per agent (ms) |

**What it does:**
1. Runs `url-scrapper` agent → saves to `shared/url.md`
2. Runs `synthesizer` agent → saves to `shared/summary.md`

**Exit codes:**
- `0` = Success
- `1` = Failure (check output for details)

## Output Files

| File | Path | Generated by |
|------|------|--------------|
| URL collection | `shared/url.md` | url-scrapper agent |
| Documentation | `shared/summary.md` | synthesizer agent |

## Related Agents

| Agent ID | Workspace | Purpose |
|----------|-----------|---------|
| `url-scrapper` | `workspace-url-scrapper` | Scrape and collect URLs |
| `synthesizer` | `workspace-synthesizer` | Synthesize into documentation |
